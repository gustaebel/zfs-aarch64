# zfs-for-aarch64
ZFS packages for Arch Linux ARM (aarch64)

## Installation

    $ cd spl
    $ makepkg -C [-c]
    $ pacman -U spl-\*.pkg.tar.xz
    $ cd ..

    $ cd zfs
    $ makepkg -C [-c]
    $ pacman -U zfs-\*.pkg.tar.xz
    $ cd ..
