# zfs-odroid-c2
ZFS packages for the ODROID-C2 running Archlinux ARM.

## Installation

    $ cd spl
    $ makepkg -C [-c]
    $ pacman -U spl-\*.pkg.tar.xz
    $ cd ..

    $ cd zfs
    $ makepkg -C [-c]
    $ pacman -U zfs-\*.pkg.tar.xz
    $ cd ..
