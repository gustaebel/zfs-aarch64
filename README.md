# zfs-odroid-c2
ZFS packages for the ODROID-C2 running Archlinux ARM.

## Installation

    $ cd spl
    $ makepkg -C [-c]
    $ sudo pacman -U spl-<version>.pkg.tar.xz
    $ cd ..

    $ cd zfs
    $ makepkg -C [-c]
    $ sudo pacman -U zfs-<version>.pkg.tar.xz
    $ cd ..
